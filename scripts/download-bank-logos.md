# Como Baixar os Logos dos Bancos

## Passo a Passo

1. **Acesse o repositório GitHub:**
   - https://github.com/Tgentil/Bancos-em-SVG

2. **Baixe os SVGs dos bancos desejados:**
   - Navegue até a pasta do banco (ex: `Nu Pagamentos S.A`)
   - Baixe o arquivo SVG (geralmente chamado `logo.svg` ou similar)

3. **Organize os arquivos na estrutura correta:**
   ```
   public/
     banks/
       Nu Pagamentos S.A/
         logo.svg
       Itaú Unibanco S.A/
         logo.svg
       Bradesco S.A/
         logo.svg
       Banco do Brasil S.A/
         logo.svg
       Banco Santander Brasil S.A/
         logo.svg
       Caixa Econômica Federal/
         logo.svg
       Banco Inter S.A/
         logo.svg
       Banco C6 S.A/
         logo.svg
       PicPay/
         logo.svg
       Mercado Pago/
         logo.svg
       PagSeguro Internet S.A/
         logo.svg
       Neon/
         logo.svg
       Banco Original S.A/
         logo.svg
       XP Investimentos/
         logo.svg
       Banco BTG Pacutal/
         logo.svg
   ```

4. **Importante:**
   - Os nomes das pastas devem corresponder EXATAMENTE aos nomes no repositório GitHub
   - O arquivo SVG deve ser nomeado como `logo.svg`
   - Se o arquivo tiver outro nome, renomeie para `logo.svg`

## Nota

Se os logos não aparecerem, verifique:
1. Se os arquivos SVG foram baixados e colocados nas pastas corretas
2. Se os nomes das pastas correspondem exatamente aos nomes no repositório
3. Se o arquivo SVG está nomeado como `logo.svg`
4. O console do navegador para verificar erros de carregamento
